# 🎓 Student Gate Pass Management System

This is a web-based application to generate and manage student gate passes.

## 🚀 Features
- Add student details
- Generate gate pass
- Store records in table
- Search by roll number
- Delete records

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
